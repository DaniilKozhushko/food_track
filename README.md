## Описание проекта:
Эта программа генерирует синтетические данные о продажах блюд в ресторане, учитывает случайные факторы (возвраты, списания, порчу) и неожиданные события (чаевые, кражи и другие рандомные происшествия). В итоге создаётся отчёт в .txt-формате с красивым табличным оформлением.

## Цели:
- отработать взаимодействие с системой контроля версий
- применить на практике навыки, приобретенные на курсах:
    - написание кода на Python
    - работа с библиотеками random, datetime
    - использование менеджера контекста для работы с файлами
- улучшить навыки форматирования текста

## Теоретические планы:
Подключить PostgreSQL, сгенерировать данные на некий период и вывести статистику (желательно в формате витрины)
